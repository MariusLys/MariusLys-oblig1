# Hello Health App

## Description
This project is a simple Python application created as part of Obligation 1.  
The purpose of the project is to verify that Python, virtual environment setup, Git, and GitHub are correctly configured before moving on to more advanced programming tasks.

## Installation Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/YOUR_USERNAME/MariusLys-oblig1.git
   cd MariusLys-oblig1
   ```
2. Create a virtual environment:
   ```bash
   python3 -m venv venv
   ```
3. Activate the venv:
   ```bash
   source venv/bin/activate
   ```
4. Run the Application: 
   ```bash
   python hello_health.py
   ```
